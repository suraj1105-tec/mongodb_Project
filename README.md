# mongodb_Project
# IBM Cognos & MongoDB Project 
 Overview

This project demonstrates the practical implementation of IBM Cognos Analytics for reporting and visualization along with MongoDB for handling NoSQL database operations. It covers report creation, data modeling, visualization techniques, and advanced database queries.
#  Objectives
Create and design reports using IBM Cognos
Visualize data using different chart types (bar, column, line, combo)
Build custom data modules for structured reporting
Apply filters, prompts, and drill-through features
Perform MongoDB operations including queries, aggregations, joins, and updates
# 📁 Project Structure
IBM Cognos Lab
Report creation from scratch
List reports and dashboards
Data visualization (charts & comparisons)
Interactive reports with prompts
Drill-through analysis
MongoDB Assignment
Database and collections creation
CRUD operations (Create, Read, Update, Delete)
Aggregation framework
Joins using $lookup
Advanced queries and filtering
